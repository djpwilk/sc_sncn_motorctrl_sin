Sinusoidal based Commutation Loop.

Input Channels c_hall channel to receive position information from hall sensor c_qei channel to receive position information from qei sensor c_signal channel for signaling after initialization of commutation loop c_commutation_p1 channel to receive motor voltage input value - priority 1 (highest) 1 ... (lowest) 3 c_commutation_p2 channel to receive motor voltage input value - priority 2 c_commutation_p3 channel to receive motor voltage input value - priority 3 c_watchdog channel to start the watchdog

Output Channels c_pwm_ctrl channel to set pwm level output to motor phases

qei_paramsthe struct defines sensor type and resolution parameters for qei
