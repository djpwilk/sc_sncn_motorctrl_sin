Parameters

qei_paramsstruct defines the resolution for qei sensor and gear-ratio
sensor_usedspecify the sensor to use via HALL/QEI defines

Input Channel c_adc channel to receive to torque information from current sensor c_hall channel to receive position information from hall c_qei channel to receive position information from qei c_torque channel to receive/send torque control information

Output Channel c_commutation channel to send motor voltage input value
