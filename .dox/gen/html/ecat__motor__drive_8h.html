This server implementation enables motor drive functions via Ethercat communication.

Input Channel pdo_in channel to receive information from ethercat coe_out channel to receive motor config information from ethercat c_signal channel to receive init ack from commutation loop c_hall channel to receive position information from hall c_qei channel to receive position information from qei

Output Channel pdo_out channel to send out information via ethercat c_torque_ctrl channel to receive/send torque control information c_velocity_ctrl channel to receive/send velocity control information c_position_ctrl channel to receive/send position control information
