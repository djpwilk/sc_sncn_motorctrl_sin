Run the watchdog timer server.

The watchdog timer needs a constant stream of pulses to prevent it from shutting down the motor. This is a thread server which implements the watchdog timer output.

The watchdog control has two differents ports attached to the watchdog circuitry. The enable signal must be the LSB bit in a 4-bit port. The tick control must be a 1-bit port.

Parameters
c_watchdogthe control channel for controlling the watchdog
p_wd_tickcontrol for the tick of the watchdog
p_shared_leds_wdencontrol port for the watchdog device
