Non triggered ADC server.

This server should be used if the somanet node is not used for motor drive/control. This is the interface to AD7949 ADC devices. It controls two devices so that two channels can be sampled simultaneously.

Parameters
c_adcchannel to receive ADC output
clkclock for the ADC device serial port
p_sclk_conv_mosib_mosia4-bit port for ADC control interface
p_data_a1-bit port for ADC data channel 0
p_data_b1-bit port for ADC data channel 1
